PKU-HRI Dataset Description
===========================
This dataset is collected from The Shenzhen Graduate School, Peking University(PKU) campus
and please refer to the PKU regulation for using it. It contains sensitive data and the 
usage should respect PKU student's privacy.
------------------------
This dataset contains 2500 pedestrian image pairs. Each pair contains two images of the 
same individual. Each image pair was taken using automatic pedestrian detector 
under varying illumination conditions. Each image has been scaled to be 128x48 pixels.
